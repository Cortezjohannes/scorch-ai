# Pre-Production Engine Mapping

## Comprehensive Engine Utilization Per Tab

This document maps all AI engines used in each pre-production tab for maximum quality output.

---

## TAB 1: NARRATIVE OVERVIEW
**Status:** Copy existing episode content (no generation)

**Engines:** None (passthrough)

**Content:** Episode summaries, scenes, branching choices from workspace

---

## TAB 2: SCRIPTS (EXPANDED)
**Generate after:** Narrative

**Engines Utilized:** 8 ENGINES

1. **DialogueEngineV2** - Rich, authentic character dialogue with subtext
2. **StrategicDialogueEngine** - Purposeful conversations that advance story
3. **PerformanceCoachingEngineV2** - Actor direction and performance notes
4. **LanguageEngineV2** - Culturally authentic dialogue and voice differentiation
5. **FiveMinuteCanvasEngineV2** - Short-form pacing optimization (5-minute episodes)
6. **TensionEscalationEngine** - Dramatic momentum and tension beats
7. **GenreMasterySystem** - Genre-specific dialogue techniques and tropes
8. **CharacterEngineV2** - Character voice consistency and 3D depth

**Output:**
- Expanded screenplay format (600-900 words per scene)
- Blocking notes: "(crosses to window)" "(sits down)"
- Performance notes: "(frustrated)" "(holding back tears)"
- Camera suggestions: "SUGGESTION: Close-up works well here" (NOT requirements)
- Room for creative interpretation and actor improvisation

**Example:**
```
INT. COFFEE SHOP - DAY

Sarah sits at corner table, phone in hand.

SARAH
(checking phone nervously)
Where is he?

SUGGESTED SHOT: Wide establishing showing both characters

MIKE
(avoiding eye contact)
Hey. Sorry I'm late.

SARAH
(stands, arms folded)
That's what you said last time.

SUGGESTED SHOT: Two-shot capturing both performances
```

## TAB 3: STORYBOARDS (WITH IMAGE GENERATION)
**Generate after:** Scripts

**Engines Utilized:** 4 ENGINES + IMAGE AI

1. **StoryboardEngineV2** - Shot sequence planning and visual continuity
2. **VisualStorytellingEngine** - Comprehensive visual framework and composition
3. **CinematographyEngineV2** - Technical camera work and lighting guidance
4. **VisualDesignEngineV2** - Color psychology and visual mood
5. **AIImageGenerator** - DALL-E 3 / Stable Diffusion for reference images

**Output:**
- Shot-by-shot breakdown (3-5 shots per scene)
- AI-generated reference image for each shot
- Shot type, camera angle, movement
- Timing estimates (flexible, not rigid)
- Lighting and mood notes
- Creative notes: "Feel free to adjust framing for emotional impact"

**Example:**
```
SCENE 2: Coffee Shop Conversation

Shot 1: ESTABLISHING WIDE (~10 min suggested)
[AI-GENERATED IMAGE]
Description: Wide shot of coffee shop interior, Sarah at table, Mike entering
Camera: Static, across room
Lighting: Natural window light
CREATIVE NOTE: Adjust framing to capture mood you feel

Shot 2: TWO-SHOT - Conversation (~15 min suggested)
[AI-GENERATED IMAGE]
Description: Medium shot, both actors at table
Camera: Straight on or slight angle (your choice)
CREATIVE NOTE: Follow the energy of the performance
```

## TAB 5: FILMING LOCATIONS (WITH IMAGE GENERATION)
**Generate after:** Props & Wardrobe

**Engines Utilized:** 3 ENGINES + IMAGE AI

1. **LocationEngineV2** - Location scouting and site analysis (if exists)
2. **WorldBuildingEngineV2** - Environmental consistency and world integration
3. **ProductionEngineV2** - Logistics, permits, and production efficiency
4. **AIImageGenerator** - Reference images for each location

**Output:**
- Location descriptions
- AI-generated reference images (16:9 cinematic)
- Logistical notes (permits, access, parking)
- Technical considerations (lighting, sound, space)
- Backup location suggestions
- Optimal shooting times

**Example:**
```
LOCATION 1: COFFEE SHOP
[AI-GENERATED IMAGE]

Description: Modern coffee shop with natural lighting, minimal decor
Episodes: 1, 3, 7, 12
Scenes: 15 total

REQUIREMENTS:
- Natural window light (shoot 9AM-2PM)
- Quiet enough for dialogue recording
- Space for tripod setup
- Accessible parking

SUGGESTIONS:
- Local independent coffee shop (not chains)
- Weekend mornings less crowded
- May need to pay location fee ($50-200)

BACKUP: Home kitchen dressed as cafe
```

## Sequential Generation Benefits

1. **Scripts use Narrative** - Better dialogue based on actual story content
2. **Storyboards use Scripts** - Visual planning based on actual dialogue and blocking
3. **Props use Storyboards** - Prop lists based on actual visual requirements
4. **Locations use Props & Storyboards** - Location needs based on visual and practical requirements
5. **Casting uses full Narrative** - Character analysis based on complete story arc
6. **Marketing uses everything** - Hooks based on actual content, not assumptions

**Result:** Each tab builds on previous work, creating a coherent, high-quality pre-production package.

---

## Creative Freedom Philosophy

All suggestions, timings, and shot lists are **GUIDES, not requirements**:

- Actors encouraged to add soul and artistry
- Timing estimates are flexible targets
- Shot suggestions can be modified on-set
- Creative intuition trumps rigid planning
- The plan serves the art, not vice versa

✨ **"Great preparation enables great improvisation"**
